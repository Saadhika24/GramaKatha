# GramaKatha

GramaKatha is a multilingual smart credit ledger Android application developed using Kotlin and Jetpack Compose. The app helps small shop owners digitally manage customer credit transactions, track outstanding balances, maintain daily reports, and send payment reminders through WhatsApp and SMS.

## Features
- Add and manage customer records
- Track due amounts and advance payments
- Daily report management
- WhatsApp and SMS payment reminders
- Search customers instantly
- Multilingual support (English, Kannada, Hindi)
- Simple and user-friendly UI

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
- Material 3
- Intent API

## Problem Statement
Small shop owners often maintain customer credit records manually in notebooks, leading to calculation errors, missing records, and difficulty in tracking payments. GramaKatha provides a digital solution for efficient customer credit management.
